🤖 EduBot – Local AI Text Generation with TinyLLaMA

EduBot is a lightweight AI text generation web application powered by the TinyLLaMA model running locally.
It provides a simple and interactive web interface built using HTML, CSS, and JavaScript, allowing users to generate AI responses without relying on external APIs or cloud services.

This project demonstrates how local Large Language Models (LLMs) can be integrated with a web application to build private, fast, and offline AI tools.

🚀 Features

🧠 Local LLM Integration using TinyLLaMA
⚡ No Internet Required – Runs completely locally
🎨 Simple UI built with HTML, CSS, and JavaScript
💬 Prompt → AI Response generation
🔒 Privacy Friendly (no external API calls)
🧩 Beginner-friendly AI project structure

🛠 Tech Stack
Technology	Purpose
TinyLLaMA	Local language model
Python	Backend logic
FastAPI	API for model interaction
HTML	Frontend structure
CSS	UI styling
JavaScript	Client-side interaction

💡 How It Works

User enters a prompt in the web interface.
The frontend sends a request to the backend API.
The backend processes the prompt using TinyLLaMA.
The generated response is sent back to the frontend.
The response is displayed in the UI.
